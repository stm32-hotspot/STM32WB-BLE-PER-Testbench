Purpose

STM32WB-BLE-PER-Testbench example aims to demonstrates how to enable BLE Direct Test Mode (DTM) in the STM32WB from a firmware user application using HCI testing commands directly in order to measure Rx PER on a STM32WB based DUT.

STM32WB-BLE-PER-Testbench example is being delivered as-is, not necessarily supported by ST.

This example includes the HAL and LL drivers provided in the STM32CubeWB_v1.13.1 package. For further information about these components, see official package.
